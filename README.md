# 🌍 Multi-Timezone Clock

A modern, responsive, and interactive **Multi-Timezone Clock** web application built using **HTML5, CSS3, TypeScript, Node.js, and Express**. The application allows users to add multiple timezone clocks, switch between 12-hour and 24-hour formats, and save their selected clocks using Local Storage.

# ✨ Features

- 🌍 Add multiple timezone clocks
- ⏰ Live clock updates every second
- 🕛 Toggle between 12-hour and 24-hour formats
- 🗑 Delete individual clocks
- 💾 Save clocks using Local Storage
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- ⚡ Fast TypeScript implementation
- ♿ Accessible form controls

# 🚀 Installation

### 1 Clone Repository

```bash
git clone https://github.com/yourusername/multi-timezone-clock.git
```

---

### 2 Open Project

```bash
cd multi-timezone-clock
```

---

### 3 Install Dependencies

```bash
npm install
```

---

### 4 Build Project

```bash
npm run build
```

---

### 5 Start Server

```bash
npm run dev
```

---

### 6 Open Browser

```
http://localhost:3000
```

---

# 🎯 How to Use

1. Select a timezone.
2. Click **Add Clock**.
3. View the live clock.
4. Toggle between **12H** and **24H** formats.
5. Delete unwanted clocks.
6. Refresh the page—saved clocks remain available.
